💡 What is a batch?

A batch is one purchase of an item. Every time you buy more stock, you add a new batch — even if it's the same product.


Example
Batch 1 → Jan 5 → Expires Mar 1
Batch 2 → Jan 12 → Expires Mar 8

System uses Batch 1 first — older stock consumed first (FIFO).

FIFO = First In, First Out. Older stock is always used before newer stock, reducing waste.
